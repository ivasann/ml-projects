# 🎬 Netflix Data Analysis — EDA

Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python.

---

## 📌 About the Project

This project explores the Netflix dataset to uncover trends and patterns in content — such as which countries produce the most content, popular genres, and how Netflix's library has grown over the years.

---

## 📂 Dataset

- **Source:** [Netflix Movies and TV Shows — Kaggle](https://www.kaggle.com/datasets/shivamb/netflix-shows)
- **File:** `netflix_titles.csv`
- **Size:** ~8,800 titles

- ✅ Insights & conclusions

---

## 📈 Key Questions Explored

1. How many Movies vs TV Shows are on Netflix?
2. Which countries produce the most content?
3. What are the most popular genres?
4. Which year had the most content added?
5. Who are the top directors on Netflix?
6. Which month does Netflix add the most content?
