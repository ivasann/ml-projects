# 🤖 AI Document Chatbot

Chat with your PDFs using AI. Upload documents, ask questions, and get accurate answers with sources.

---

## 🚀 Features

* Multi-PDF support
* Chat-style interface
* Source citations
* Fast responses (Groq)

---

## 🛠️ Tech

Streamlit • LangChain • FAISS • HuggingFace • Groq

---

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 🔑 API Key

Enter your Groq API key inside the app.

## 💡 Use Cases

Research • Business docs • Study assistant
