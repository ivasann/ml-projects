# Antitrust RAG System

Simple Retrieval-Augmented Generation (RAG) system for querying a DOJ vs Google antitrust case document.

## Features
- Semantic search over PDF
- Context-based answers
- Modular pipeline

## Usage
```bash
python app.py
```

## Stack
- Python
- LangChain
- FAISS / Chroma
- LLM APIs
